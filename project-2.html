Moon Rabbit Cafe

WEB DESIGN / BRANDING / WORDPRESS

Using Wordpress and Divi, this project is a website created for a fictional dessert cafe for a school assignment to simulate building a website for a client.

Problem

The goal of this assignment was to create a responsive website using Wordpress with the Divi builder for a fictional client in the form of a restaurant. The website had to include menu and about pages, a consistent nav bar, and the images had to be optimized for efficient loading.

Outcome

A branded responsive website with breakpoints for desktop, tablet, and mobile created for a Chinese dessert cafe with optimized images using Wordpress and Divi.

Home Page

In developing the cafe's branding, the visual design was inspired by the blue and white of porcelain plates, which also inspired the rounded corners. This cafe aims to provide a relaxing space for people to visit and enjoy tea and desserts.

Menu Page

The menu has a simple layout and menu with the cafe's featured items with pictures at the top, and other items categorized below.

About Page

Moon Rabbit Cafe's mission statement is inspired by the Chinese myth of the rabbit in the moon that assists the moon godess by making medicine. The logo is inspired by motifs of the moon, rabbits, and blue and white porcelain plates.

Contact Page

Page contains a contact form where patrons can give feedback to the business.
